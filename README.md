# 国家卫健委标准 BMI+体脂率 综合计算器（2024开源版）

[![GitHub license](https://img.shields.io/badge/license-LGPL--3.0-blue.svg)](https://github.com/your-username/bmi-calculator/blob/main/LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/your-username/bmi-calculator)](https://github.com/your-username/bmi-calculator/stargazers)

一个基于国家卫健委《体重管理指导原则（2024年版）》的BMI+体脂率综合计算器，支持全年龄段（0-100岁+）人群评估。

## 🌟 核心特性

- 📊 **权威标准**：严格依据国家卫健委2024年最新标准开发
- 👥 **全年龄段覆盖**：
  - 0-7岁儿童（月龄别SD评估）
  - 7-18岁儿童青少年（年龄别筛查）
  - 18-65岁成人（BMI+腰围）
  - 65岁以上老年人（适宜范围）
  - 孕妇（孕前BMI评估）
- 🔢 **多指标综合评估**：
  - BMI指数计算
  - 体脂率估算
  - 中心型肥胖（腰围）判定
  - 营养状况分类
- 📱 **响应式设计**：适配移动端和桌面端
- 🔗 **分享功能**：生成可分享的结果链接
- 🚀 **零依赖**：纯HTML+CSS+JavaScript，无需后端

## 📖 数据来源

本计算器依据以下国家卫健委官方标准开发：

| 标准名称 | 文件编号 | 适用人群 |
|---------|---------|---------|
| 《体重管理指导原则(2024年版)》 | 最新标准 | 全人群 |
| 《成人体重判定》 | WS/T 428-2013 | 18岁及以上成人 |
| 《学龄儿童青少年超重与肥胖筛查》 | WS/T 586-2018 | 7-18岁儿童青少年 |
| 《7岁以下儿童生长标准》 | WS/T 423-2022 | 0-7岁儿童 |
| 《妊娠期妇女体重增长推荐值标准》 | WS/T 801-2022 | 孕妇 |
| 《国民体质测定标准（2023年修订）》 | 最新标准 | 国民体质 |
| 《中国居民膳食指南（2022）》 | 最新标准 | 膳食指导 |

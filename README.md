### 📋 标准引用说明

1. **权威性**：所有标准均来自中华人民共和国国家卫生健康委员会（NHC）官方网站
2. **时效性**：采用2022-2024年发布的最新版本标准
3. **完整性**：计算器严格遵循标准中的计算公式、界值和建议
4. **透明性**：每个标准均提供可直接访问的官方链接

### ⚠️ 使用注意事项

1. **标准版本**：请确保使用最新版本标准，本计算器已更新至2024年最新标准
2. **适用范围**：不同标准针对不同人群，请根据用户年龄选择相应标准
3. **数据权威**：所有计算依据均为国家卫健委官方发布的标准文件
4. **标准更新**：医学标准会定期更新，请关注官方发布的最新版本
